## ElectroBudget
A budgetting app running on Electron. 

## Usage
This project is intended to be used on a wide monitor and in dark mode. 

## Available Scripts

In the project directory, you can run:

### `yarn electron:start`
Concurrently runs `yarn start` and `electron .`.
